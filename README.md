# MARS OpsFlow

- Internal operations dashboard concept for a cosmetics business workflow.
- Built around MARS Cosmetics internship responsibilities: dashboards, automation, SOP support, reporting, and AI-assisted drafting.
- Live demo: https://harshit1330.github.io/mars-ops-ai-dashboard/

## Features

- Workflow queue for inventory, support, marketing, and operations tasks.
- Automation suggestions for spreadsheet, API, CRM, and reporting workflows.
- SOP search to quickly find process guidance by keyword.
- Weekly report summary with copy-to-clipboard support.
- AI-style draft generator for vendor, support, and SOP update messages.

## Tech Stack

- HTML
- CSS
- JavaScript
- GitHub Pages

## Run Locally

- Open `index.html` directly in a browser.
