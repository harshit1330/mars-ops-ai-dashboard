# MARS OpsFlow

MARS OpsFlow is a static internal operations dashboard concept built for the MARS Cosmetics internship brief.

It demonstrates:

- Internal web tools and dashboards
- Workflow queues for repetitive business tasks
- Automation recommendations for spreadsheets, APIs, and alerts
- SOP search for internal process assistance
- AI-style drafting for vendor, support, and SOP updates
- Reporting summaries for business teams

## Tech

- HTML
- CSS
- JavaScript
- Local static data

## Run

Open `index.html` in a browser, or visit it through the portfolio GitHub Pages deployment.
