# MARS OpsFlow

MARS OpsFlow is an internal operations dashboard concept for a cosmetics business workflow. It is built around workflow queues, automation suggestions, SOP support, reporting, and AI-assisted drafting for business teams.

## Features

- Workflow queue for inventory, customer support, marketing, and operations tasks.
- Automation suggestions for spreadsheet, API, CRM, and reporting workflows.
- SOP search for process guidance by keyword.
- Weekly report summary with copy-to-clipboard support.
- AI-style draft generator for vendor follow-ups, support escalations, and SOP update notes.
- Metrics cards for open items, SLA health, data alerts, and SOP coverage.
- Responsive static dashboard layout.

## Install and Run Locally

No package installation is required.

```bash
git clone https://github.com/harshit1330/mars-ops-ai-dashboard.git
cd mars-ops-ai-dashboard
```

Open `index.html` directly in a browser, or serve the folder with any static server.
